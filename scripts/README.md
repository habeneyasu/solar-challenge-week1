# Scripts Directory

This directory contains utility scripts for the Solar Data Discovery project.

## Scripts

### `example_oo_usage.py`

Demonstrates the usage of Object-Oriented patterns implemented in the project.

**Purpose**: Show how to use the OO classes (`SolarDataAnalyzer`, `DataQualityAssessor`, `CorrelationAnalyzer`, and visualization classes) for data analysis.

**Usage**:
```bash
cd scripts
python example_oo_usage.py
```

**Features Demonstrated**:
- Object-Oriented data analysis
- Strategy pattern for different analysis methods
- Template Method pattern for visualizations
- Inheritance pattern for specialized visualizers
- Encapsulation of data and methods
